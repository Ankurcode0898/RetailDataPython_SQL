# RetailDataPython_SQL
# 🛒 Retail Data Analysis using Python and SQL

## 📌 Project Overview

This project focuses on analyzing retail sales data using **Python**, **Pandas**, and **SQL** to extract meaningful business insights.  
The dataset contains retail order information such as sales, profit, category, region, shipping mode, and customer details.

The main objective of this project is to:
- Clean and preprocess raw retail data
- Perform exploratory data analysis (EDA)
- Analyze sales and profit trends
- Execute SQL queries for business insights
- Build a strong beginner-friendly Data Analytics project

This project is useful for learning:
- Python for Data Analysis
- SQL for Business Analysis
- Data Cleaning Techniques
- Real-world Retail Analytics Workflow
